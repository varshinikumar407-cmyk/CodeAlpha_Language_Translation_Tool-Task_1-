#  Language Translator using Python

## Internship Task 1 Project

This project is a simple Language Translator built using Python.  
It takes text input from the user and translates it into another language
using the Google Translate API (googletrans library).

---

## Features
- Accepts user input from the terminal
- Automatically detects the source language
- Translates text into the selected target language (Spanish)
- Displays translated text clearly in the terminal

---

## Technologies Used
- Python 3
- googletrans library
- Visual Studio Code

---

## How to Run the Project

1. Install Python on your system
2. Install required library: 
3. Open the project folder in Visual Studio Code
4. Open terminal inside the folder
5. Run the program:
6. Enter the text when prompted
7. View the translated output in the terminal
